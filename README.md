# My Learning Journal
A responsive, clean, and modern blog-style website designed to document a coding bootcamp journey. This project focuses on modern CSS layouts, semantic HTML, and responsive design principles.

## 🚀 Live Demo
For live demo, please navigate to: https://mylearningjournalblog.netlify.app/

## ✨ Features

- **Responsive Header**: A sticky navigation bar that adapts to different screen sizes using CSS clamp().

- **Hero Banner**: A high-impact hero section with a dark overlay and bottom-aligned typography to ensure readability.

- **Intelligent Grid**: A blog post layout powered by CSS Grid's auto-fit and minmax, enabling a fully responsive layout without excessive media queries.

- **Modern Typography**: Utilizes the "Source Sans 3" Google Font with fluid scaling for optimal readability.

- **Elegant UI**: Includes text truncation for post descriptions and hover effects for interactive elements.

## 🛠️ Technical Highlights
- **CSS Layouts**: Flexbox for the header and banner; CSS Grid for the post list.

- **Fluid Design**: Heavy use of the clamp() function for font sizes, heights, and padding.

- **Visual Enhancements**:

  - `object-fit`: cover for consistent blog post image ratios.
  - `linear-gradient` overlays for banner text contrast.

  - Multi-line text truncation using `-webkit-line-clamp`.
